1. Alohida component uchun style
Fayl nomi: _ComponentName.scss

Izoh: Bu SCSS fayl faqat bitta React/Next komponentga tegishli bo‘ladi.

Import qilish: Faqat komponent ichida import qilinadi.

Misol:

Komponent: Card.tsx

Style: _Card.scss


// Card.tsx ichida
import "./_Card.scss";



2. Global style uchun
Fayl nomi: componentName.scss (prefix yo‘q)

Izoh: Bu fayl global yoki sahifa darajasida qayta-qayta ishlatiladigan style'lar uchun.

Import qilish: globals.scss yoki layout.tsx ichida import qilinadi.

Misol:

Global style: card.scss (barcha kartochkalar uchun umumiy dizayn)

scss
// globals.scss ichida
@import "./styles/card";
3. Kerakli tartib:
Har bir komponent uchun o‘zining _ComponentName.scss fayli bo‘ladi.

Umumiy qayta ishlatiladigan style’lar esa componentName.scss faylida saqlanadi.

components/, pages/, styles/ papkalari ajratilgan bo‘ladi.
